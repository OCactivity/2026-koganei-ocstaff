# 法政大学 小金井オープンキャンパススタッフ 公式サイト

法政大学 小金井キャンパスのオープンキャンパススタッフ（OCスタッフ）の  
**活動内容・各チーム紹介・イベント情報・新規メンバー募集**を目的とした公式Webサイトです。

---

## 📌 サイト概要

本サイトは、以下の目的で制作されています。

- 小金井オープンキャンパススタッフの活動内容を高校生・新入生に分かりやすく伝える
- 各チーム（CM / CT / IS / SC / SK）の役割や魅力を紹介する
- 年間イベントや人気行事を写真付きで紹介する
- 新規スタッフ募集への導線を用意する

---

## 🧑‍🤝‍🧑 各チーム紹介

- **CMチーム**：  
  広報・企画を中心としたチーム
- **CTチーム**：  
  当日の運営・誘導を担うチーム
- **ISチーム**：  
  来場者対応・案内を担当
- **SCチーム**：  
  ステージ・イベント進行を担当
- **SKチーム**：  
  全体統括・サポート業務

※ 詳細はサイト内「各チーム紹介」ページを参照してください。
